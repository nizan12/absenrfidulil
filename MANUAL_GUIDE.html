Bab 01

Panduan untuk Siswa

Cara melakukan absensi menggunakan kartu RFID

A Cara Absensi

1

Siapkan Kartu RFID

Setiap siswa memiliki kartu RFID yang telah didaftarkan. Pastikan kartu selalu dibawa setiap hari.

2

Tempelkan Kartu ke Alat

Tempelkan kartu ke perangkat pembaca RFID di gerbang sekolah dengan jarak maksimal 3 cm. Tunggu hingga terdengar bunyi beep.

3

Cek Layar LCD

Layar LCD akan menampilkan nama Anda beserta status: "SELAMAT DATANG" (masuk) atau "SAMPAI JUMPA" (pulang).

4

Notifikasi Terkirim

Orang tua/wali akan otomatis menerima notifikasi WhatsApp bahwa Anda telah masuk atau pulang sekolah.

B Aturan Tap

Tap ke-StatusKeterangan
1 (pertama hari ini)MASUKSaat tiba di sekolah
2KELUARSaat pulang sekolah
3, 4, ...BergantianSelalu bergantian MASUK / KELUAR
Jangan Tap Terlalu Cepat

Setiap tap memiliki jeda waktu (default 5 menit). Jika tap terlalu cepat, layar akan menampilkan "TUNGGU DULU" dan absensi tidak akan tercatat.

C Jika Kartu Bermasalah

  • Jika muncul "KARTU TIDAK TERDAFTAR" — lapor ke guru piket atau admin
  • Jika kartu tidak terbaca sama sekali — coba tempelkan ulang lebih dekat
  • Jika kartu hilang atau rusak — segera lapor ke Staff Admin untuk penggantian

Bab 02

Panduan untuk Orang Tua

Menerima notifikasi dan memantau kehadiran anak

A Notifikasi WhatsApp Otomatis

Setiap kali anak Anda melakukan tap di sekolah, Anda akan menerima notifikasi WhatsApp otomatis dengan format:

Contoh Notifikasi Masuk

📍 Notifikasi Kehadiran

Ananda Ahmad Fauzan telah MASUK sekolah pada:
📅 15 Mei 2026
⏰ 07:15 WIB
📍 Gerbang Utama

SMAIT Ulil Albab

B Cek Kehadiran via Website

Anda juga dapat melihat riwayat kehadiran anak tanpa login:

1

Buka Website

Akses https://absenulilalbab.com di browser HP atau komputer.

2

Cari Nama Anak

Ketik nama anak di kotak pencarian pada halaman utama.

3

Lihat Riwayat

Klik nama anak dari daftar. Anda dapat melihat status kehadiran hari ini dan riwayat sebelumnya.

Pastikan Nomor HP Benar

Notifikasi dikirim ke nomor WhatsApp yang terdaftar di sistem. Jika tidak menerima notifikasi, hubungi pihak sekolah untuk memastikan nomor HP sudah benar dan opsi notifikasi aktif.

Bab 03

Panduan untuk Guru Piket

Monitoring kehadiran harian dan input manual

A Login ke Dashboard

1

Buka Website

Akses https://absenulilalbab.com dan klik tombol Masuk.

2

Masukkan Kredensial

Isi email dan password yang diberikan oleh admin, lalu klik Login.

B Menu yang Tersedia

MenuFungsi
DashboardStatistik kehadiran hari ini (jumlah hadir, belum hadir)
Live MonitorPantau data tap secara real-time tanpa refresh
RekapitulasiLihat rekap kehadiran berdasarkan tanggal dan kelas

C Input Manual (USB RFID Reader)

Untuk siswa yang kartu RFID-nya bermasalah:

1

Buka Halaman Utama

Di halaman landing (tanpa login), gunakan fitur Manual Tap.

2

Scan atau Ketik UID

Gunakan USB RFID reader atau ketik UID manual.

3

Tekan Enter

Data akan langsung terkirim ke server dan tercatat sebagai absensi.

B Menambah Siswa Baru

1

Buka Menu Data Siswa

Klik Data Siswa di sidebar.

2

Klik Tambah Siswa

Klik tombol + Tambah Siswa di pojok kanan atas.

3

Isi Data

Lengkapi: RFID UID, NIS, Nama, Kelas, Kategori, dan foto (opsional).

4

Simpan

Klik Simpan. Siswa siap melakukan tap absensi.

Pendaftaran Kartu RFID

Semua kartu RFID didaftarkan oleh administrator sistem. Perangkat di gerbang hanya digunakan untuk presensi, bukan untuk membaca nomor kartu.

B Import Data Massal

1

Download Template

Buka template, isi data sesuai kolom. Jangan ubah header.

3

1

2

Tambah Perangkat

Panduan untuk Super Admin

Akses penuh ke seluruh fitur sistem

A Fitur Eksklusif Super Admin

Menu EksklusifFungsi
Data AdminMembuat, mengedit, dan menghapus akun pengguna/admin
PengaturanKonfigurasi sistem: info institusi, API Key, Fonnte Token, tap delay

Super Admin juga memiliki akses ke semua menu yang dimiliki Staff Admin, Operator, dan Guru Piket.

B Menambah Admin/Pengguna

1

Buka Data Admin

Klik menu Data Admin di sidebar.

2

Klik Tambah

Klik + Tambah Admin. Isi: Nama, Email, Password, No Telepon.

3

Pilih Role

Pilih role: super_admin, kepala_sekolah, staff_admin, guru_piket, atau operator.

4

Simpan

Klik Simpan. Pengguna baru bisa langsung login.

C Pengaturan Sistem

1

Buka Pengaturan

Klik menu Pengaturan di sidebar.

2

Konfigurasi

Atur: Nama Institusi, Logo, API Key ESP32, Fonnte Token, dan Tap Delay (jeda antar tap).

3

Simpan Perubahan

Klik Simpan. Perubahan langsung berlaku.

API Key Harus Sinkron

Pastikan API Key di Pengaturan sama persis dengan API_KEY di kode ESP32. Jika berbeda, perangkat tidak bisa mengirim data tap.

Troubleshooting & Tips

Solusi masalah umum dan tips penggunaan

A Masalah Umum & Solusi

MasalahSolusi
Kartu RFID tidak terbacaTempelkan ulang lebih dekat (maks 3 cm). Pastikan kartu tidak rusak fisik.
LCD menampilkan "KARTU TIDAK TERDAFTAR"Kartu belum didaftarkan di sistem. Hubungi admin.
LCD menampilkan "TUNGGU DULU"Masih dalam jeda waktu (5 menit). Tunggu beberapa saat.
LCD menampilkan "DEVICE ERROR"Perangkat belum terdaftar/nonaktif di server. Hubungi admin.
LCD menampilkan "WIFI FAILED"Koneksi WiFi bermasalah. Restart perangkat ESP32.
LCD menampilkan "RFID ERROR / CEK KABEL"Modul RFID tidak terhubung. Periksa kabel dan restart ESP32.
Tidak menerima notifikasi WACek nomor HP di sistem. Pastikan WhatsApp aktif dan Fonnte Token valid.
Website error / tidak bisa diaksesCoba refresh (Ctrl+F5). Jika masih error, hubungi admin.
Lupa passwordHubungi Super Admin untuk reset password.

B Tips Penggunaan

  • Simpan kartu RFID di tempat yang aman, hindari kartu terlipat atau basah
  • Tap saat datang dan saat pulang setiap hari agar data kehadiran lengkap
  • Jangan meminjamkan kartu RFID ke orang lain
  • Orang tua: pastikan nomor WhatsApp yang terdaftar selalu aktif
  • Admin: lakukan backup data secara berkala
Butuh Bantuan?

Hubungi tim IT sekolah atau Super Admin untuk bantuan lebih lanjut terkait sistem absensi RFID.
